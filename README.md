# gomamayo

ゴママヨを検知します。

## 使い方

```
cargo run ゴママヨ 太鼓公募募集終了 オレンジジュース
ゴママヨ: 1項1次のゴママヨです。
太鼓公募募集終了: 3項2次のゴママヨです。
オレンジジュース: ゴママヨではありません。
```
